# Alexander_Oncology
 This is a Python-based full-stack web application designed to manage and visualize cancer oncology patient data on a dashboard fully capable of interactivity. It consists of a PostgreSQL database tier, an indexing data processor, a mock data generation engine, and is all aided by Plotly Dash. A standalone script utilizing the Faker library alongside Python's native random module. A mix of faker and the random module helps populate the remote database with 500 mock patient records with realistic clinical classifications (like Melanoma, Stage IV, Immunotherapy etc.) mapped to coordinate areas across the entirety of the continental United States.There is also a basic PostgresSQL based SQL that requires username and password check in order to gain access to the information once the web app opens.
